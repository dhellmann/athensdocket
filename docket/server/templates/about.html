This site uses the following tools:

  • Python for all programming.

  • Celery for task distribution
  • RabbitMQ for messaging
  • Fabric for deployment automation
  • Twitter Bootstrap for CSS and JavaScript framework
  • GitHub for source code hosting, bug tracking, and project management
